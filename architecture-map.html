Package map

bindkit/ |-- cmd/server/ # main: load -> wire -> serve |-- internal/ | |-- mcp/ # protocol, registry, dispatch | |-- server/ # pipeline + stdio + http | |-- auth/ # key -> principal | |-- ratelimit/ # token bucket per key | |-- metering/ # memory + redis counters | |-- billing/ # quota + Stripe stubs | |-- config/ # typed, validated env | `-- logging/ # slog + redaction |-- tools/ | `-- example_weather/ # copy this shape |-- deploy/ # Dockerfile / fly / compose |-- docs/ # clients / pricing / launch |-- scripts/new_tool.sh # make new-tool `-- Makefile / CI / go.mod

coremcp

JSON-RPC framing, Handler interface, concurrent tool registry, and dispatcher for initialize, tools/list, and tools/call.

coreserver

Glues middleware around the dispatcher and serves over stdio or HTTP with graceful shutdown.

gateauth / ratelimit / billing

Resolve API keys, throttle per key, and enforce monthly quota before expensive tool execution.

inframetering / config / logging

Usage counters, env validation that fails loudly, and structured logs with secret redaction.
